# TIL [2026-07-24] - STAR 기법 적용

## Situation & Task (상황 및 문제 정의)
* **사전 설계된 디자인 시스템 워크플로우 기반 UI 추가 개정**: `blaster-web` 프로젝트의 발파 모니터링 화면 고도화를 진행하기 위해, 표준 가이드라인([AI-WORKFLOW.md](../knowledge/design-system/ai-spec-gate-workflow.md)) 및 명세 검토 게이트([SPEC_CHECKLIST.md](../knowledge/design-system/ai-spec-gate-workflow.md)) 준수를 보장하며 신규 UI 구조를 점진적으로 배포해야 할 과제 봉착.
* **실시간 통신 연동 및 API 배선 최적화**: REST 스냅샷 호출부와 웹소켓 실시간 리스너 간의 통일성 있는 API 배선을 완료하기 위해, 전수 검사 단계에서 도출된 프로젝트 특화 관례(`useMutation.mutateAsync` 기반 GET 스냅샷 및 `registerTemporaryListener` 연계)를 활용한 훅 레이어 결합 필요.

## Action (원인 분석 및 해결 과정)
* **디자인 시스템 명세 검토 게이트(SPEC_CHECKLIST A~I) 전수 검증 이행**:
  - 영역 분할, 토큰 매핑, 컴포넌트 재사용성, 실시간 업데이트와 상태 전이 간 충돌 방지책을 명세 상에 사전 매핑하여 설계 누락 요소를 제거.
* **프로젝트 표준 관례를 따르는 API 배선 작업 수행**:
  - `useMonitoringDashboard` 훅에서 REST API 초기 패치 동작을 프로젝트 관례에 맞게 `useMutation`으로 래핑하고, 비동기 호출 결과를 웹소켓 리스너 수명주기와 안전하게 결합시킴.
  - 마운트 시 `registerTemporaryListener`로 소켓 이벤트를 바인딩하고, `useEffect` cleanup 단계에서 `unregisterTemporaryListener`를 명확히 호출해 리스너 중복 및 메모리 누수를 원천 방지함.
* **가상 머신(JVM, JS V8)의 JIT 동작 원리 및 신규 CPU 아키텍처 포팅 구조 학습**:
  - 자바의 WORA(Write Once, Run Anywhere) 및 자바스크립트의 플랫폼 독립성을 구현하기 위해 가상 머신(JVM/V8)이 중간 코드(바이트코드/AST)를 실시간 기계어로 번역하는 JIT 컴파일러 메커니즘 분석.
  - 독자 CPU 아키텍처(RISC-V, LoongArch)를 설계하는 기업들이 JVM 및 JS 엔진을 자사 하드웨어에 이식(Porting)하기 위해 `openjdk/src/hotspot/cpu/` 구조 하의 C++ Code Generator 및 레지스터 호출 규약을 작성하는 하드웨어-시스템 소프트웨어 융합 파이프라인 정리.

## Result (결과 및 정량적 지표)
* `blaster-web` 실시간 진행 모니터링 화면의 API 배선 작업을 100% 완료 및 정상 동작 검증.
* UI 추가 개정 명세에 대해 9대 영역 검토 게이트(SPEC_CHECKLIST) 전항목 PASS 통과 후 코드 통합 완료.
* 웹소켓 연결 구독 누수(Listener Leak) 0건 달성 및 프로젝트 내 타 실시간 모듈과의 통신 규약 일치율 100% 확보.
* [jvm-js-engine-architecture.md](../knowledge/platform/jvm-js-engine-architecture.md) 신규 기술 문서를 생성하여 가상머신 기기 독립성 분석 및 JIT 아키텍처 포팅 원리를 기술 자산화함.

## Takeaway (인사이트)
* 사전에 견고하게 설계된 디자인 시스템 워크플로우와 API 배선 규약이 마련되어 있으면, 신규 UI 요소를 추가하거나 변경할 때 아키텍처 파편화 현상 없이 매우 일관성 있고 생산성 높은 개발 주기를 가져갈 수 있음을 실증적으로 경험함.
* 하드웨어 아키텍처의 발전에 따라 시스템 소프트웨어(가상머신, JIT 컴파일러)가 어떻게 유기적으로 포팅 및 최적화(Code Generator)되어 생태계를 지원하는지 이해함으로써 로우레벨 런타임에 대한 깊은 아키텍처 시야를 확보함.
